// Beginner's Guide — How to Use
What is this?
This simulator shows you exactly what happens inside DES and AES encryption — step by step with visual tables. See every bit permutation, every S-box lookup, every XOR as it happens in real time.
DES — Step by Step
1
Plaintext — Type any 8 characters. Example: HELLO123. DES works on 64 bits = 8 ASCII characters exactly.
2
Key — Type any 8 characters as your secret key. Example: SECRETKY. DES uses 56 of these 64 bits (8 parity bits are dropped by PC-1).
3
Mode — Encrypt converts plaintext → ciphertext. Decrypt reverses it. For decrypt, paste the 16-char hex from a previous encryption as input.
4
Simulate — Output appears instantly. Use Next/Prev to step through, or AUTO to play. Press ⏸ Pause anytime to freeze and study a step.
Test Decrypt: Encrypt HELLO123 key SECRETKY → copy Output Hex → switch Decrypt → paste → same key → Simulate → see HELLO123 recovered!
AES — Step by Step
1
Plaintext — Type exactly 16 characters. AES works on 128-bit blocks = 16 bytes. Example: HelloWorldCrypto.
2
Key Size:  AES-128 = 16 chars (fast, common)  |  AES-192 = 24 chars  |  AES-256 = 32 chars (strongest).
3
Key — Match the key size. Example AES-128: SecretKey1234567 (exactly 16 chars). This is your password.
4
Simulate — Steps show every round: SubBytes (S-box grid), ShiftRows (row diagram), MixColumns (column table), AddRoundKey (XOR table).
Test Decrypt: Encrypt something → copy Output Hex (32 chars) → switch Decrypt → paste hex → same key size & key → Simulate → ASCII shows original text!
Playback Controls
▶
AUTO — Plays all steps. ⏸ Pause appears while playing — click it to freeze any step. Then Resume, Skip to End, or Stop.
#
Jump — Type a step number and click Go to jump directly to any step instantly.
⚡
Speed — Slow / Normal / Fast / Very Fast for auto-play.
⎘
Copy — Click the Copy button next to Output Hex to copy the result for use in decrypt mode.
